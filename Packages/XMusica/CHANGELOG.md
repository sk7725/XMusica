# Changelog

## [1.2.0] - 2023-07-15
### Changes
- In-editor piano displays now detect mouse hover.

## [1.1.0] - 2023-07-15
### Changes
- VInst editor windows now dock, preferring other VInst windows and then the scene view.

### Additions

- Added a Virtual Instrument tester window.
